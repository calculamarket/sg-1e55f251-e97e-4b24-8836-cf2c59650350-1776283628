# Dashboard Marketplaces

## Vision
Dashboard de acompanhamento de vendas para Mercado Livre e Shopee. Permite visualizar métricas consolidadas, pedidos recentes, e configurar credenciais de API dos marketplaces. Voltado para vendedores que operam em múltiplas plataformas.

## Design
Cores (HSL format):
- `--primary: 158 64% 52%` (verde vibrante - #10B981)
- `--secondary: 217 91% 60%` (azul - #3B82F6)
- `--background: 0 0% 100%` (branco)
- `--foreground: 222 47% 11%` (dark slate)
- `--accent: 217 91% 60%` (azul)
- `--muted: 220 13% 91%` (cinza claro)
- `--card: 0 0% 100%` (branco)
- `--border: 220 13% 91%` (cinza claro)

Fontes:
- Heading: Plus Jakarta Sans (600, 700)
- Body: Work Sans (400, 500, 600)

Estilo: Clean, data-driven, profissional. Cards elevados, hierarquia clara, badges coloridos para status.

## Features
- Dashboard principal: métricas consolidadas (vendas totais, pedidos, ticket médio), gráficos de performance
- Listagem de pedidos recentes com filtros por marketplace
- Página de configurações: formulários para API keys do Mercado Livre e Shopee
- Sistema de notificações para novos pedidos (preparação futura)
